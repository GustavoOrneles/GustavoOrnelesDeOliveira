Projetos em destaque

De ideias inspiradas a projetos proprios, aqui estão alguns dos meus trabalhos mais recentes

  • Galeria de imagens Jquery

    Este projeto é uma galeria de imagens interativa desenvolvida com jQuery. A galeria permite aos usuários adicionar novas imagens copiando o endereço de qualquer imagem e colando-o em um campo específico. Após a adição, a imagem aparece na galeria com efeitos de fade, proporcionando uma transição suave e agradável entre as imagens. Com um design intuitivo e responsivo, a galeria é fácil de usar em dispositivos de todos os tamanhos. A integração do jQuery permite uma interação suave e dinâmica com a galeria, tornando a experiência do usuário mais agradável e envolvente. Este projeto é uma excelente demonstração do uso do jQuery para criar funcionalidades interativas em páginas da web, oferecendo aos usuários uma maneira simples e elegante de visualizar e adicionar novas imagens à galeria.

  • Próximo Projeto: Loja virtual de jogos

    Este projeto é um site de compra de jogos semelhante ao launcher da Epic Games, com todas as funcionalidades de uma plataforma de distribuição digital de jogos inspirado em plataformas populares como a Epic Games Store, o site oferecerá uma ampla seleção de jogos para os usuários comprarem e adicionarem às suas bibliotecas. Desenvolvido com React Bootstrap para o frontend e incluindo funcionalidades de backend para envio de emails e mensagens personalizadas, adicionar e remover items do carrinho, o site promete uma experiência completa e intuitiva para os usuários. Com um design moderno e responsivo, o site será fácil de usar em dispositivos de todos os tamanhos, oferecendo uma experiência de compra de jogos online eficiente e agradável. Este projeto está sendo construído do zero e promete ser uma adição emocionante ao mundo das plataformas de distribuição digital de jogos. "
